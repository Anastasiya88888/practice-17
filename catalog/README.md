# Практичне заняття 17. Розгортання додатка

## Додаток дозволяє:

- Переглядати персонажів у вигляді карток
- Додавати нових персонажів
- Імпортувати персонажів з онлайн-API
- Завантажувати та кешувати зображення
- Переглядати статистику

## Файли проєкту

| Файл               | Опис                                 |
|--------------------|--------------------------------------|
| `app.py`           | Графічний інтерфейс                  |
| `main.py`          | Консольна версія каталогу            |
| `api_client.py`    | Робота з API                         |
| `image_manager.py` | Завантаження та кешування зображення |
| `character.json`   | База персонажів                      |
| `requirements.txt` | Список залежностей                   |

## Функції GUI

- Перегляд персонажів у вигляді карток
- Відкриття деталей персонажа
- Додавання нового персонажа
- Імпорт з API (із завантаженням зображень)
- Перегляд статистики
- Можливість видалити персонажів
