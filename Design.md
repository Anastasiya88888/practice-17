# Дизайн-документ: Каталог персонажів

## 1. Опис проєкту
Консольний додаток для управління каталогом персонажів (ігрових персонажів, героїв тощо).

## 2. Функціональні вимоги

### 2.1 Основні функції:
- Перегляд списку всіх персонажів
- Додавання нового персонажа
- Перегляд деталей персонажа
- Збереження даних

## 3. Структура даних

### 3.1 Персонаж (Character):
- `id` - унікальний ідентифікатор
- `name` - ім'я персонажа
- `type` - тип персонажа (воїн, маг, лучник)
- `health` - здоров'я
- `attack` - сила атаки
- `imageUrl` - посилання на зображення
- `local_image_path` - Шлях до локального зображення

## 4. Список команд

| Команда         | Опис                               |
|-----------------|------------------------------------|
| `list, ls`      | Показати всіх персонажів           |
| `add, create`   | Додати нового персонажа            |
| `show <id>`     | Показати деталі персонажа          |
| `import, fetch` | Імпортувати персонажів з API       |
| `cache`         | Показати статистику кешу зображень |
| `clear-cache`   | Очистити кеш зображень             |
| `help, ?`       | Показати довідку                   |
| `exit, quit`    | Вийти з програми                   |


## 5. Приклад використання
```
> list

2. Мерлін (Маг) - HP: 60, ATK: 80

> show 1

```
## 6. Функції (графічного додатка):
- Перегляд всіх персонажів у вигляді карток
- Клік по картці відкриває деталі
- Кнопка "Оновити" - перезавантажує список
- Кнопка "Імпорт" - завантажує нових персонажів
- Кнопка "Статистика" - показує інформацію про кеш

## 7. Розгортання

### 7.1 Ціль
Забезпечити можливість запуску додатка без встановлення Python і бібліотек.

### 7.2 Метод розгортання та результат
За допомогою PyInstaller:
``` 
pyinstaller --noconsole --onefile app.py 
```
Створено файл:
```
dist/app.exe
```
який можна запускати на будь-якому Windows-комп'ютері.
